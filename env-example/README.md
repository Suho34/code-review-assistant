# Environment Variable Examples

This folder contains a template for the environment variables required to run Mockmind AI.

## Usage

1. Copy the `.env.example` file to the root of the project and rename it to `.env`.
2. Replace the placeholder values with your actual credentials.

```bash
cp env-example/.env.example ../.env
```
